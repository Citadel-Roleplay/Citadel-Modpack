# Use the following for switching dragon genes with multiple templates.
# In order to synchronise the appearance of dragons as characters, and the dragonriders when we display them riding their dragon, we must bypass the vanilla gene system almost entirely.
# We are bypassing everything, aging, inheritable genes, etc. Game script assigns gene values, and we then use portrait and trait portrait modifiers to display them.
# When a dragon is created it is assigned a set of variables in 00_agot_dragon_templates.txt, that file contains ranges of gene strengths and their probabilities just like ethnicity files normally do.
# Straightforward genes with only one gene template are assigned in dragon_trait_modifiers.txt. As far as I (Buck) can tell, we cannot have weighted random selection in trait portrait modifiers.
# Therefore more complicated genes with multiple templates will use this file for selecting a template. A value for a template will be assigned in 00_agot_dragon_templates.txt, then in this file the corresponding template will be set.
# These are always followed by a version of the gene for the rider.
